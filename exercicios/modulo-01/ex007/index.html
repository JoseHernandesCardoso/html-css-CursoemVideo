Principais Formatações em HTML5

Negrito / Destaque

Existem duas formas de se criar um texto em negrito

A primeira é usando a tag B, que não é semantica nem recomendada

A segunda é a tag STRONG, que é semântica e mais recomendada

Itálico / Ênfase

Também existe uma forma semântica e não semantica de usar o itálico

A forma não semântica é usando a tag I

A forma semântica e mais recomendada é usando a tag EM

Texto Marcado

Podemos marcar um texto usando a tag MARK

Texto GRANDE e pequeno

Usamos a tag BIG para deixarmos o texto maior, mas está obsoleta e não é recomendado o usá-la

Já a tag SMALL, nós usamos para criar textos menores, porém, por algum motivo, ela não está obsoleta e pode ser usada normalmente (vai entender)

Texto Excluído e Inserido

A tag DEL marca um texto como excluído

Mas a tag INS marca o texto como inserido

A tag U também deixa o texto sublinhado, mas não é semântica

Texto sobrescrito e subscrito

Podemos escrever coisas como x2 com o SUP

E para fazermos H2O, usamos SUB
