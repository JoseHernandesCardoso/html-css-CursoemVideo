Algumas Outras Formatações em HTML5

Códido-Fonte e Pré-Formatação

Podemos marcar um código usando a tag CODE assim: print('Hello, world!)

Mas, para não estragar a formatação do nosso código, devemos usar a tag PRE, para deixar a formatação da forma que foi escrita, veja:

        
    num = int(input('Digite um número: '))
    if num % 2 == 0:
        print(f'O número {num} é par!)
    else:
        print(f'O número {num} é impar!')
    print('FIM DO PROGRAMA!')
